int GameCore.Resource.ResourceManager.Destroy ( GameObject  go)

게임 오브젝트 제거 함수

Parameters
go제거할 오브젝트
Returns
0 정상 작동, -1 에러 발생, 1 정상 작동(오브젝트 폴링 적용)

게임오브젝트를 제거하는 예시입니다.

GameObject go = new GameObject("ObjectName");
int i = Managers.Resource.Destroy(go);
switch (i)
{
case -1:
Debug.Log("에러 발생")
break;
case 0:
Debug.Log("정상 작동")
break;
case 1:
Debug.Log("정상 작동(오브젝트 폴링)")
break;
}

T리소스의 타입

리소스의 타입에 대해

  • txt, json파일은 TextAsset 타입
  • 텍스쳐로 사용할 png파일은 Texture2D 타입
  • sprite로 사용할 png파일은 Sprite 타입
  • 사운드 mp3등의 파일은 AudioClip타입

리소스를 로드하는 예시입니다.

var hitSound = Managers.Resource.Load<AudioClip타입>("sound/hit.mp3");
